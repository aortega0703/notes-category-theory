\subsection*{Diagonal functor}
For a category $C$, the diagonal functor $\Delta: C\to C\times C$ consists of
maps of: \parencite{awodey:category_theory}
\begin{itemize}
  \item Objects:\\
    $\Delta(c) = (c, c),\ (\forall c\in C)$
  \item Morphisms:\\
    $\Delta(f) = (f, f),\ (\forall f\in C)$
\end{itemize}
